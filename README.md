# Huawei Theme — HarmonyOS NEXT (寰宇)

Custom Huawei/HarmonyOS theme with extensive modifications.

## Structure
- `theme/` — Packed `.hwt` theme file
- `apk-reversed/` — Reversed source of Cosmos SuperWallpaper APK

## Theme Details
- **Name:** HarmonyOS NEXT (寰宇)
- **Version:** 14.0.10 (EMUI 14 / HarmonyOS)
- **Author:** AbobaHostl
- **Screen:** FHD (1080p)
- **Accent:** Day `#317AF7` (blue), Night `#E75746` (red-orange)

## Features
- Custom lockscreen with media controls (bipbip engine)
- 16 weather clock widgets (4x2 grid)
- 214 custom app icons
- Always On Display support
- Complete framework theming (day/night)
- 16 system module overrides
- Live wallpaper (Cosmos SuperWallpaper)
- DroidSansChinese font

## Modules
- framework-res-hwext: System accent colors
- com.android.systemui: Status bar & control center
- com.huawei.android.launcher: Home screen
- com.huawei.aod: Always On Display
- com.huawei.calendar: Calendar app
- com.huawei.deskclock / com.android.deskclock: Clock app
- com.android.settings: Settings app
- com.huawei.camera: Camera UI
- com.huawei.music / com.android.mediacenter: Music app
- com.huawei.systemmanager: System manager
- com.huawei.security.privacycenter: Privacy center
- com.huawei.vassistant: Voice assistant
- com.huawei.intelligent: AI assistant
- com.huawei.android.hwouc: Theme manager
- com.huawei.android.totemweather.skinrespck: Weather app
